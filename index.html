🔬 Model Specification & Computational Framework

Methodological foundations, LaTeX equations, and algorithmic specifications of the Markov cohort simulation

1. Markov Cohort State transitions

The cohort is modeled monthly across 5 mutually exclusive health states. The state distribution vector is defined as:

\mathbf{S}_t = [S_{t,\text{PF}}, S_{t,\text{NonOp}}, S_{t,\text{Opioid}}, S_{t,\text{CPSP}}, S_{t,\text{Death}}]

State occupancy at month t+1 is calculated via matrix multiplication with transition matrix \mathbf{P}:

\mathbf{S}_{t+1} = \mathbf{S}_t \times \mathbf{P}

The transition probability matrix is structured as follows:

\mathbf{P} = \begin{pmatrix} p_{\text{PF}\to\text{PF}} & p_{\text{PF}\to\text{NonOp}} & 0 & 0 & p_{\text{PF}\to\text{Death}} \\ p_{\text{NonOp}\to\text{PF}} & p_{\text{NonOp}\to\text{NonOp}} & p_{\text{NonOp}\to\text{Opioid}} & p_{\text{NonOp}\to\text{CPSP}} & p_{\text{NonOp}\to\text{Death}} \\ p_{\text{Opioid}\to\text{PF}} & p_{\text{Opioid}\to\text{NonOp}} & p_{\text{Opioid}\to\text{Opioid}} & p_{\text{Opioid}\to\text{CPSP}} & p_{\text{Opioid}\to\text{Death}} \\ p_{\text{CPSP}\to\text{PF}} & 0 & 0 & p_{\text{CPSP}\to\text{CPSP}} & p_{\text{CPSP}\to\text{Death}} \\ 0 & 0 & 0 & 0 & 1.0 \end{pmatrix}

Opioid reduction efficacy (\theta_{\text{VR}}) reduces the baseline risk of developing chronic pain (R_{\text{CPSP}}) for patients transitioning from the opioid state:

p(\text{Opioid} \to \text{CPSP})_{\text{VR}} = R_{\text{CPSP}} \times (1 - \theta_{\text{VR}})

2. Amortized Costs & QALY Weighting

The total cost of the VR therapy intervention per patient (C_{\text{VR}}) is amortized over the lifetime patient capacity of the system (N_{\text{Patients}}) and incorporates session cost overhead:

C_{\text{VR}} = C_{\text{Base}} + \frac{C_{\text{Capital}}}{N_{\text{Patients}}} + ( C_{\text{Session}} \times N_{\text{Sessions}} )

Quality-Adjusted Life Years (\text{QALYs}) are computed by summing the time spent in each health state multiplied by the state utility weights (u_s), adjusted by a monthly fractional factor and discount rate (r):

\text{QALYs} = \sum_{t=1}^{12} \left( \sum_{s \in \mathcal{S}} S_t(s) \times u_s \right) \times \frac{1}{12} \times \frac{1}{(1 + r)^{t/12}}

3. Cost-Effectiveness Decision Metrics

The Incremental Cost-Effectiveness Ratio (ICER) represents the additional cost per QALY gained by adopting VR therapy compared to Standard Care:

\text{ICER} = \frac{\Delta C}{\Delta E} = \frac{C_{\text{VR}} - C_{\text{SC}}}{E_{\text{VR}} - E_{\text{SC}}}

The Net Monetary Benefit (NMB) converts QALY gains into currency terms using the Willingness to Pay threshold (\lambda):

\text{NMB} = (\lambda \times \Delta E) - \Delta C

4. Societal & Productivity Scaling Projections

Direct health system savings are scaled up to a regional cohort of size N_{\text{Cohort}}:

S_{\text{Health}} = (C_{\text{SC}} - C_{\text{VR}}) \times N_{\text{Cohort}}

Productivity losses from workplace absenteeism are modeled based on state occupancy weights A_s \in \{0.20, 0.50, 0.80\} for states s \in \{\text{NonOp}, \text{Opioid}, \text{CPSP}\}. The annual weeks of absenteeism saved is computed as:

\Delta W_{\text{Abs}} = \sum_{t=1}^{12} \left[ \sum_{s \in \mathcal{S}} (S_{t,\text{SC}}(s) - S_{t,\text{VR}}(s)) \times A_s \right] \times 4.33 \times N_{\text{Cohort}} \times R_{\text{Emp}}

Workplace productivity economic savings are then calculated based on average weekly earnings W_{\text{Earn}}:

\text{Savings}_{\text{Productivity}} = \Delta W_{\text{Abs}} \times W_{\text{Earn}}

Informal caregiver hours saved are computed using care overhead coefficients H_s \in \{8, 20, 20\} hours/month:

\Delta H_{\text{Caregiver}} = \sum_{t=1}^{12} \left[ \sum_{s \in \mathcal{S}} (S_{t,\text{SC}}(s) - S_{t,\text{VR}}(s)) \times H_s \right] \times N_{\text{Cohort}}

5. Computational Execution Algorithms

Below are the pseudocode specifications for the core Markov iteration loop, stochastic sensitivity modeling, and numerical root-finding algorithms.

Algorithm 1: Markov Cohort Simulation Loop

Input: Parameter profile Θ, Time Horizon T = 12 months, Initial Vector S0
Output: Cumulative QALYs E, Cumulative Healthcare Costs C

1: Initialize state occupancy: S0 ← [0.0, 0.3, 0.7, 0.0, 0.0]
2: Apply efficacy adjustments: P(Opioid → CPSP) ← Risk × (1 - Efficacy)
3: Initialize accumulators: E ← 0, C ← 0
4: For month t = 0 to T-1:
5:     Compute transition matrix multiplication: St+1 ← St × P
6:     Calculate QALY increment: E ← E + Σs St+1(s) × us × (1/12)
7:     Calculate healthcare cost: C ← C + Σs St+1(s) × Costs × (1/12)
8: End For
9: Add intervention acquisition cost to VR Arm cost: C ← C + CVR
10: Return E, C

Algorithm 2: Probabilistic Sensitivity Analysis (PSA)

Input: Model parameters baseline Θ, Monte Carlo Iterations N = 200
Output: Scatter plot points { (ΔEi, ΔCi) } for i = 1 to N

1: For iteration i = 1 to N:
2:     Sample utilities: us,i ← BetaDistribution(mean = us, SD = 0.1 × us)
3:     Sample unit costs: Costs,i ← GammaDistribution(mean = Costs, SD = 0.2 × Costs)
4:     Construct stochastic parameters Θi
5:     Run Algorithm 1 on Standard Care: (ESC,i, CSC,i) ← Simulate(Standard, Θi)
6:     Run Algorithm 1 on VR Therapy:   (EVR,i, CVR,i) ← Simulate(VR, Θi)
7:     Compute differences: ΔEi ← EVR,i - ESC,i,   ΔCi ← CVR,i - CSC,i
8:     Save scatter point: PlotData.push( (ΔEi, ΔCi) )
9: End For
10: Return PlotData

Algorithm 3: One-Dimensional Threshold Solver

Input: Objective function f(x) = NMB(x) - NMBtarget, search bounds [a, b], tolerance ε = 0.0001
Output: Root threshold value x* (where Net Monetary Benefit = 0)

1: Initialize bounds: x0 ← a, x1 ← b
2: Calculate objective values: y0 ← f(x0), y1 ← f(x1)
3: While |x1 - x0| > ε:
4:     Calculate secant step: xnext ← x1 - y1 × (x1 - x0) / (y1 - y0)
5:     Bound next step to search constraints: xnext ← Clamp(xnext, a, b)
6:     Update states: x0 ← x1,  x1 ← xnext
7:     Evaluate target: y0 ← y1,  y1 ← f(x1)
8:     If |y1| < tolerance: Break
9: End While
10: Return x1
